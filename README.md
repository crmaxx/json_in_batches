# JsonInBatches

Library for streamed JSON.

## Installation

Add this line to your application's Gemfile:

    gem 'json_in_batches'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install json_in_batches

## Usage

    Product.where("...").to_json_in_batches
