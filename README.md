# JsonInBatches

Library for streamed JSON.

## Installation

Add this line to your application's Gemfile:

    gem 'json_in_batches', :git => 'git://github.com/crmaxx/json_in_batches.git'

And then execute:

    $ bundle

## Usage

    Product.where("...").to_json_in_batches
